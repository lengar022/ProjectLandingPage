IoT model of in-car sensor and telemetry system displayed on screen and in the cloud

Abstract

A telemetry system that collects and displays real-time data from a rally car has been developed. A Raspberry Pi acts as the central controller, interfacing with several sensors including a GPS module, a DS18B20 temperature sensor, a fuel level sender measured through an ADS1115 analogue-to-digital converter, and BLE TPMS sensors. The Raspberry Pi processes the sensor data and publishes it to an MQTT broker. A webpage built using Svelte subscribes to this data stream. The dashboard presents live sensor readings alongside the vehicle’s real-time location on a map, allowing users to remotely monitor the car’s status and location.
